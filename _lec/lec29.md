# Logic Programming in miniKanren

## Setup 
 1. Install a DrRacket (>= 8.0)
 2. `Language>Choose Language>The Racket Language>OK`
 3. `File>Package Manager>'faster-minikanren'>install`
 4. Make sure your file begins with the following:
 
 ```
 #lang racket
 (require minikanren)
 ```
